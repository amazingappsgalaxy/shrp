## Portrait vs Landscape Mode
- portrait=true (default): Uses skin-enhancement workflow with node #341 and #337 active
- portrait=false: Uses alternative workflow, #341 and #337 not overridden

## Resolution (Max Mode only)
- Max mode OFF: output resolution determined by workflow
- Max mode ON + 4K: #378 width=4096, height=4096 → label "4K Sharp"
- Max mode ON + 8K: #378 width=8192, height=8192 → label "8K Ultra"

## Implementation Notes
- Workflow selection is done via getProUpscalerWorkflow(portrait, maxmode) helper
- skinPreset sent from frontend as string '1'|'2'|'3' (from Subtle/Real/Cinema selection)
- Runs through startTaskForModel() in runninghub-provider.ts
